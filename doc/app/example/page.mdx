# Example app

The repo includes a **runnable Node example** at [`examples/basic/`](https://github.com/cyphra/cyphra/tree/main/examples/basic) in the GitHub tree.

It demonstrates:

- Parsing and printing a `.cyphra` schema
- Parameterized Cypher with the **`cypher`** tag and **`SelectQuery`**
- Offline **DDL preview** for `cyphra push` — constraints and range indexes
- Optional **live** `RETURN 1` via **`CyphraClient.queryRecords`** when `NEO4J_*` env vars are set

The example depends on the **`cyphra`** meta-package only (`import { … } from "cyphra"`).

## Run locally

From the monorepo root:

```bash
pnpm install
pnpm build
pnpm example:basic
```

See the example [`README.md`](https://github.com/cyphra/cyphra/blob/main/examples/basic/README.md) for details and `.env` setup.
