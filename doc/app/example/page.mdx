# Example project

The Cyphra monorepo ships a **minimal Node.js example** at [`examples/basic/`](https://github.com/cyphra-dev/cyphra/tree/main/examples/basic) on GitHub.

It demonstrates:

- Parsing and printing a **`.cyphra`** schema
- **Parameterized Cypher** with the **`cypher`** tag and **`SelectQuery`**
- Offline **DDL preview** (constraints + range indexes) aligned with **`cyphra push`**
- Optional **live** `RETURN 1` via **`CyphraClient.queryRecords`** when **`NEO4J_*`** env vars are set

The example depends only on the **`cyphra`** meta-package (`import { … } from "cyphra"`).

## Run locally

From the repository root:

```bash
pnpm install
pnpm build
pnpm example:basic
```

See [`examples/basic/README.md`](https://github.com/cyphra-dev/cyphra/blob/main/examples/basic/README.md) for **`.env`** setup and details.
