# CLI

| Command                          | Description                   |
| -------------------------------- | ----------------------------- |
| `cyphra init`                    | Scaffold config and schema    |
| `cyphra migrate deploy`          | Run pending migrations        |
| `cyphra migrate dev`             | Same with verbose logging     |
| `cyphra migration:create <name>` | New `.mjs` migration          |
| `cyphra push`                    | Apply constraints from schema |
| `cyphra generate`                | Reserved                      |

Requires `NEO4J_URI`, `NEO4J_USER`, `NEO4J_PASSWORD`.
