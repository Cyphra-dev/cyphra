# Cyphra

Syntax highlighting and language support for **`.cyphra`** files — Cyphra’s graph schema language for modeling **Neo4j** data in your repository.

## Install

Install from the [**Visual Studio Marketplace**](https://marketplace.visualstudio.com/items?itemName=Cyphra.cyphra-vscode): open the **Extensions** view in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) and search for **Cyphra**.

## Features

- Syntax highlighting for the Cyphra schema DSL (`node`, `relationship`, decorators, types, …)
- Comment and bracket behavior tuned for `.cyphra` files
