# @cyphra/runtime

Default **runtime** entry for Cyphra on Neo4j: re-exports provider types from **`@cyphra/core`** and **`CyphraClient`** / **`CyphraNeo4j`** from **`@cyphra/provider-neo4j`**, plus helpers to map query results to plain objects.

## Install

```bash
npm install @cyphra/runtime
```

Equivalent imports are available from the **`cyphra`** meta-package if you prefer one dependency.
