# @cyphra/core

Shared **graph provider** types for Cyphra: datasource identifiers, session/transaction shapes, and the driver contract used by **`@cyphra/migrator`** and provider packages such as **`@cyphra/provider-neo4j`**.

## Install

```bash
npm install @cyphra/core
```

Most apps depend on **`cyphra`** or **`@cyphra/runtime`** instead of importing **`@cyphra/core`** alone.
